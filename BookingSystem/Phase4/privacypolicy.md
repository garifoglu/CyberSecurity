# Privacy Policy

## 1. Introduction
This Privacy Policy explains how personal data is collected, used, and protected within the Web-based Booking System. The system complies with the General Data Protection Regulation (GDPR).

## 2. Personal Data We Collect
We may collect the following personal data:
- Name
- Email address
- Username
- Age (to verify legal eligibility)
- Booking-related information

## 3. Purpose of Data Processing
Personal data is processed for:
- User registration and authentication
- Booking management
- System administration and security
- Legal compliance

## 4. Legal Basis for Processing
Data is processed based on:
- User consent
- Contractual necessity
- Legal obligations

## 5. Data Storage and Security
Personal data is securely stored and protected against unauthorized access. Passwords are hashed using secure algorithms.

## 6. Data Retention
Personal data is retained only as long as necessary for the purposes stated above.

## 7. User Rights
Users have the right to:
- Access their personal data
- Request correction or deletion
- Withdraw consent at any time

## 8. Contact
For privacy-related questions, please contact the system administrator.
